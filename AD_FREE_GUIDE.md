## Recommended Ad Blockers

### 1. **uBlock Origin** (Best Option)
   - **Chrome/Edge**: [Install from Chrome Web Store](https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm)
   - **Firefox**: [Install from Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/)
   - **Safari**: [Install from App Store](https://apps.apple.com/app/adguard-for-safari/id1440147259)

### 2. **AdBlock Plus**
   - Available for all major browsers
   - [Download here](https://adblockplus.org/)

### 3. **Brave Browser**
   - Built-in ad blocker
   - No extension needed
   - [Download Brave](https://brave.com/)

## Additional Tips

### For Mobile Users:
- **iOS**: Use Brave Browser or Safari with AdGuard
- **Android**: Use Brave Browser or Firefox with uBlock Origin

### For Desktop Users:
- Install uBlock Origin extension
- Enable all filter lists in settings
- Add custom filters if needed

### If You Still See Ads:
1. Try switching to a different server
2. Clear browser cache and cookies
3. Make sure your ad blocker is enabled
4. Try a different browser with ad blocking
